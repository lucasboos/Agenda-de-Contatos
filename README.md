# About
A simple Python Web Application using Django.

## Setup

**Create the virtual environment**
```
python -m venv venv
```

**Activate the virtual environment**
*Windows*
```
venv\Scripts\activate
```
*Linux / Mac OS*
```
source venv/bin/activate
```

**Packages installation**
```
pip install django
```

**Running**
```
python manage.py runserver
```
